# StudyManager - Study Timer & Learning Tracker App

**学習の記録を、はじめよう。**

StudyManager は「計画 → 実行 → 振り返り」を一つの画面で管理するシンプルで美しい学習管理アプリです。

## 機能

- **タイマー機能**: 科目ごとに学習時間を計測
- **科目管理**: 複数の科目を登録・管理
- **学習ログ**: 学習履歴を自動記録
- **統計表示**: 本日の学習時間、セッション数、集中度を表示
- **ローカル保存**: ブラウザのローカルストレージに自動保存

## 使い方

1. **科目を選択**: 学習する科目を選択します
2. **タイマーを開始**: 「Start」ボタンをクリックして計測開始
3. **学習を完了**: 「Finish」ボタンで記録を保存
